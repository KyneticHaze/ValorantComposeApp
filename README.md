# Valorant Hub

> Valorant information application built using the Valorant API

## Project Features
- 100% Kotlin based
- Minimum sdk 28
- Jetpack Compose
- MVVM with Clean Architecture
- Flow
- Navigation Components
- Hilt for Dependency Injection
- Retrofit
- Coil
